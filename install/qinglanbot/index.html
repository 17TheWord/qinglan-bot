# 前提条件

前提条件

青岚 Bot 基于 NoneBot2open in new window 开发，因此依赖于 Pythonopen in new window。
Python 版本请选择 Python3.9 / Python3.10（Python3.8 暂未测试）
不会安装 Python 的可以参考 廖雪峰的教程open in new window。

警告

NoneBot2open in new window 与 NoneBotopen in new window 不兼容且无法共存！如果你曾经使用过 NoneBot 或基于 NoneBot 开发的机器人（如：yobotopen in new window、HoshinoBotopen in new window、dd-botopen in new window），或者你不确定是否安装过 NoneBot。

那么在开始安装之前，请先尝试在终端（Powershell/cmd）内执行 pip uninstall nonebot 卸载 NoneBot，或者创建一个 虚拟环境open in new window 使 青岚 Bot 可以与 NoneBot 共存。

如果你的 Python 基础较为薄弱，推荐使用 脚手架安装
